System Requirements

The distribution constructor requires the following setup:

Table 1-1 System Requirements

Requirement

Description

OpenSolaris 2008.11

The OpenSolaris 2008.11 release must be installed on your system. For installation instructions, see Getting Started With OpenSolaris 2008.11.

Alternatively, install the OpenSolaris development build 98, or later, from http://www.opensolaris.org/os/downloads/.

Note - In order to use the distribution constructor, you must have root access on your system.

Required packages

The SUNWdistro-const package, containing the distribution constructor application, must be installed on your system.

The distribution constructor requires the following additional packages:

  • SUNWipkg

  • SUNWpython-cherrypy

  • SUNWpython-pyopenssl

  • SUNWinstall-libs

  • SUNWinstall

You can use the Package Manager tool to install the required packages. For an introduction to the Package Manager, see this screencast.

Alternately, use IPS commands such as the following to install these packages:

$ pfexec pkg install SUNWdistro-constr

Network access

Your installed system must have network access.

Specifically, distribution constructor accesses an IPS repository that is available on the network, in order to retrieve packages for the ISO image. You must have network access to the repository that you specify in the manifest file.
