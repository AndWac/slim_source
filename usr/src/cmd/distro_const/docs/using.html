Using the Distribution Constructor

Caution - This document is in development for an upcoming release.

Prerequisites: If you have not used the distribution constructor before, read Chapter 1, Introduction to the Distribution Constructor. Then, install the application on your system, using the instructions at Chapter 2, Setup for the Distribution Constructor.

Once you have installed the distribution constructor application on your system, you are ready to use this tool to design and build custom images. There are three basic steps in this process.
