What is the Distribution Constructor?

The distribution constructor is a command-line tool for building preconfigured, bootable OpenSolaris images. This tool takes an XML manifest file as input, and builds an ISO image that is based on the parameters specified in the manifest file. Optionally, a USB image can be created, based on the generated ISO image.

An ISO image or a USB image is a collection of software that comprises an entire operating system in a single file. These images are bootable and are usable for installation, for creating a bootable CD or DVD, and other purposes. An ISO image can be made available for distribution from the Internet. A USB image can be copied to a USB flash drive.

Note - The Distribution Constructor outputs a USB image that could work in other types of flash memory devices, if that device has driver support provided by the OpenSolaris release. The usbcopy utility can be used to copy a USB image and is available with the OpenSolaris release.

The default image created by the distribution constructor is comparable to the Live CD image that's distributed as the OpenSolaris 2009.06 release. However, you can modify the content of the ISO image, selecting and adding additional packages to the image. You can revise the default settings for the resulting booted environment to create a custom, redistributable ISO image or USB image.

For x86 image installation instructions, see Getting Started With OpenSolaris 2009.06. Or, starting with the OpenSolaris 2009.06 release, you can build a SPARC image by using the automated installer tool. For SPARC automated installation instructions, see the OpenSolaris 2009.06 Automated Installer Guide.
