System Requirements

The Distribution Constructor requires the following setup:

Table 1-1 System Requirements

Requirement

Description

OpenSolaris 2008.11

The OpenSolaris 2008.11 release must be installed on your system. For installation instructions, see Getting Started With OpenSolaris 2008.11.

Alternatively, install the OpenSolaris development build 98, or later, from http://www.opensolaris.org/os/downloads/.

Note - In order to use the Distribution Constructor, you must have root access on your system.

SUNWdistro-const package

The SUNWdistro-const package, containing the Distribution Constructor application, must be installed on your system.

You can use the Package Manager tool to install the SUNWdistro-const package. For an introduction to the Package Manager, see this screencast.

Alternately, use the following IPS command to install this package:

$ pfexec pkg install sunwDistro-Constr

PythonTM

The Distribution Constructor requires the Python programming language.

Download the current version of Python at http://www.python.org/download/.

Network access

Your installed system must have network access.

Specifically, Distribution Constructor accesses an IPS repository that is available on the network, in order to retrieve packages for the ISO image. You must have network access to the repository that you specify in the DC manifest file.
