What is the Automated Installer?

The automated installer is a tool that provides simplified setup and configuration for multiple installations of the OpenSolaris operating system. The automated installer enables system administrators to create an installation service that provides blueprints for specific x86 or, starting with OpenSolaris 2009.06 release, SPARC installations. This installation service includes a web server, which stores a list of manifest files with installation specifications that have been made available by the administrator. Client machines can contact the web server and review the available services with associated manifest files, in order to find an installation blueprint that matches the client's machine specifications. When a matching blueprint, or manifest file, is found, the service installs the client with the OpenSolaris release, according to the specifications in the manifest files.
