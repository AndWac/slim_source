Downloading the Image Questions

Where do I find the AI image that I need for x86? for SPARC?

Discussion Points:

Go to http://genunix.org/. (Add explanation of the various images that are available at this site.)

How do I modify the manifest file to point to the repository of my choice?

Modify the following field in the manifest file:

<ai_pkg_repo_default_authority>
<main url="http://pkg.opensolaris.org" authname="opensolaris.org"/>
</ai_pkg_repo_default_authority>

Where do I find the images that can be used as the boot image for automated installations?

The images are located in the following locations:

  • For the default x86 release, go to http://pkg.opensolaris.org.

  • For a SPARC image or the latest development x86 images, go to http://pkg.opensolaris.org/dev.

Note - If you are a Sun employee, and you want the default release, go to http://ipkg.sfbay. Or, for the latest development build, go to http://ipkg.sfbay/dev/en/index.shtml.
