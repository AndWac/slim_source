Chapter 1

Introduction to the Automated Installer

Caution - This document is in development for an upcoming release.

Starting with the OpenSolaris 2008.11 release, the automated installer application enables system administrators to install an x86 OpenSolaris operating system on multiple platforms, using new enhanced and simplified installation tools.

Note - Starting with the OpenSolaris 2009.06 release, you can choose to install a SPARC image or an x86 image on multiple systems by using the automated installer tool.

  • If you are not familiar with the automated installer, review the following introduction to this tool. Then, see the instructions for Chapter 2, Automated Installer Setup Instructions.

  • If you are already familiar with the automated installer and have set up an install server, go to Chapter 3, How to Administer an Automated Install Server.
