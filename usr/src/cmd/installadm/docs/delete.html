How to Delete Install Services

Use the following command to delete an install service from your system:

installadm delete-service [-x] svcname

This command removes the specified image and makes free disk space available. The svcname is a required field that specifies the name of the image to be removed.

Note - Without the -x option, the installadm delete-service command deletes the image, but does not make the free space available.
