How to Enable and Disable Install Services

You can start and stop services on a system.

Enable a Service

Use the following command to start or restart a specified service.

installadm enable svcname

The svcname field is required to specify the name of the service to be started.

This command also starts or restarts the web server associated with the service.

Disable a Service

Use the following command to stop a specified service.

installadm disable svcname

The svcname field is required to specify the name of the service to be stopped.

This command also stops the web server associated with the service.
