Automated Installer Files

When you have installed the SUNWinstalladm-tools package, the following application files are available on your system.

Table 1-2 Files Table

File Location

Description

usr/lib/installadm/create-client

usr/lib/installadm/delete-client

usr/sbin/installadm

usr/lib/installadm/installadm-common

These files contain the installadm utility, which is the core utility for the automated installer application. These files also contain the tools for creating and deleting custom installer clients.

usr/lib/installadm/setup-dhcp

usr/lib/installadm/setup-image

usr/lib/installadm/setup-service

usr/lib/installadm/setup-tftp-links

These files contain components of the installadm utility that are used to set up the entities that are needed by the installer, including setting up an installation image, an installer service, and DHCP service.

/usr/share/man/man1m/installadm.1m

/usr/share/doc/auto_install/index.html

These files contain documentation for the automated installer, including the following:

  • installadm(1M) man page

  • OpenSolaris 2008.11 Automated Installer Guide

    Note - This guide is also available at http://dlc.sun.com/osol/docs/content/2008.11/AIinstall/.
